# Room Agent - 회의실 예약 자동화

Slack `/qw` 명령어로 다우오피스 회의실 예약을 자동화하는 봇입니다.

## 기능

- 자연어로 회의실 조회 및 예약 요청
- 11층 회의실 빈 시간대 조회
- 예약 시 11층 → 10층 → 9층 순서로 자동 탐색
- Google Calendar 연동 (선택사항)

## 설치 방법

### 1. 의존성 설치

```bash
pip install -r requirements.txt
```

### 2. 설정 파일 확인

`config.yaml` 파일이 이미 설정되어 있습니다. 필요시 수정하세요.

> **슬래시 커맨드 설정 필수**
> `config.yaml`의 `slack.command`와 `slack-app-manifest.json`의 `slash_commands[].command`를 동일한 값으로 채워야 합니다.
>
> ```yaml
> # config.yaml
> slack:
>   command: /qw   # 사용할 슬래시 커맨드 (예: /qw)
> ```
>
> ```json
> // slack-app-manifest.json
> "slash_commands": [{ "command": "/qw", ... }]
> ```
>
> Slack API 대시보드(api.slack.com)에서 앱의 Slash Commands 설정도 동일하게 맞춰야 합니다.

### 3. 다우오피스 로그인 (최초 1회)

```bash
python login.py
```

브라우저가 열리면 다우오피스에 로그인하세요. 로그인 후 쿠키가 `cookies.json`에 저장됩니다.

### 4. Google Calendar 연동 (선택사항)

```bash
python google_auth.py
```

브라우저에서 Google 계정 인증을 완료하면 `token.json`이 생성됩니다.

## 실행 방법

```bash
python -m src.main
```

시스템 트레이에 아이콘이 나타나고 Slack 명령어를 받을 준비가 됩니다.

## 사용 방법 (Slack)

### 회의실 현황 조회 (시간 미지정)

시간 없이 날짜만 입력하면 11층 회의실의 빈 시간대를 보여줍니다.

```
/qw 내일
/qw 오늘 회의실
/qw 내일 조회
```

응답 예시:
```
📋 2026-02-05 11층 회의실 현황

11-1 (6인)
빈 시간: 09:00-12:00(3시간), 14:00-18:00(4시간)

11-2 (6인)
빈 시간: 09:00-18:00(9시간)

11-3 (10인)
빈 시간: 09:00-10:00(1시간), 13:00-18:00(5시간)

11-4 (8인)
빈 시간: 09:00-11:00(2시간), 15:00-18:00(3시간)
```

### 회의실 조회 + 예약 (시간 지정)

시간을 지정하면 해당 시간에 빈 회의실을 보여주고 예약 버튼이 표시됩니다.

```
/qw 내일 1시
/qw 오늘 오후 2시부터 3시까지
/qw 2월 5일 10시 30분
/qw 내일 오전 10시 팀미팅
```

응답 예시:
```
📋 2026-02-05 11층 회의실 현황

11-1 (6인)
빈 시간: 09:00-12:00(3시간), 14:00-18:00(4시간)
...

────────────────
13:00-14:00 예약 가능:
[11-1 예약] [11-2 예약] [11-3 예약]
```

### 시간 표현 예시

| 입력 | 해석 |
|------|------|
| `1시` | 13:00 (오후 1시) |
| `오전 10시` | 10:00 |
| `오후 3시` | 15:00 |
| `10시 30분` | 10:30 |
| `2시~4시` | 14:00-16:00 |

### 날짜 표현 예시

| 입력 | 해석 |
|------|------|
| `오늘` | 오늘 날짜 |
| `내일` | 내일 날짜 |
| `모레` | 모레 날짜 |
| `2월 5일` | 2026년 2월 5일 |
| `다음주 월요일` | 다음주 월요일 |

### 반복 예약

매주 특정 요일에 같은 시간으로 예약하려면:

```
/qw 매주 월요일 10시 팀미팅
/qw 매주 금요일 2시~3시 스탠드업
```

- 월~금 요일만 지원
- 기본적으로 해당 월 말일까지 반복
- Google Calendar 체크 시 반복 일정(RRULE)으로 등록

**예약 모달에서 변경 가능:**
- 반복 요일 (월~금 드롭다운)
- 반복 시작일 (오늘 이후만 선택 가능)
- 반복 종료일

### 점심 후 스터디 예약

점심시간 기반으로 스터디/미팅을 예약하려면:

```
/qw 점심 11:45 스터디
/qw 점심 12:00 팀미팅
/qw 밥 12:30 스터디
```

- 점심시간(1시간) 종료 후 바로 시작하는 1시간 예약
- 30분 단위로 올림하여 시작 시간 결정
- 자동으로 **매주 반복 예약**으로 처리 (오늘 요일 기준)
- 모달에서 반복 종료일 변경 가능

| 입력 | 점심 종료 | 예약 시간 |
|------|----------|----------|
| `점심 11:45` | 12:45 | 13:00~14:00 |
| `점심 12:00` | 13:00 | 13:00~14:00 |
| `점심 12:15` | 13:15 | 13:30~14:30 |
| `점심 12:30` | 13:30 | 13:30~14:30 |

**특정 월 지정:**
```
/qw 4월 점심 1시 스터디
/qw 3월 밥 12:30 팀미팅
```

- 해당 월의 첫 번째 해당 요일부터 월말까지 반복
- 예: `4월 점심 1시` → 4월 첫 번째 해당 요일 ~ 4월 30일

**예약 모달에서 변경 가능:**
- 반복 요일 (월~금 드롭다운 선택)
- 반복 시작일 (오늘 이후만 선택 가능)
- 반복 종료일

인식 키워드: `점심`, `밥`, `밥시간`, `점심후`, `밥후`

## 예약 흐름

1. `/qw 내일 2시 팀미팅` 입력
2. "🔍 회의실 확인 중..." 표시
3. 11층 회의실 현황 + 예약 버튼 표시
4. 원하는 회의실 버튼 클릭
5. 회의 이름 입력 모달 + Google Calendar 등록 체크박스
6. "⏳ 예약 중..." DM 전송
7. 채널에 완료 메시지 공유
8. DM으로 상세 확인 (Calendar 등록 여부 포함)

### Google Calendar 등록 옵션

예약 모달에서 "Google Calendar에 등록" 체크박스로 캘린더 등록 여부를 선택할 수 있습니다.

- 체크 상태는 다음 예약 시에도 유지됩니다 (config.yaml에 저장)
- 체크 해제하면 그룹웨어만 예약되고 캘린더에는 등록되지 않습니다

## 회의실 우선순위

조회 시 11층만 표시, 예약 시 빈 방 없으면 다른 층 탐색:

1. **11층** (Tier 1): 11-1, 11-2, 11-3, 11-4
2. **10층** (Tier 2): 10-3, 10-4, 10-6
3. **9층** (Tier 3): 9-1, 9-3, 9-4

## 트러블슈팅

### 로그인 만료

세션이 만료되면 **자동으로 처리**됩니다:

1. Slack DM으로 "🔐 그룹웨어 세션이 만료되었습니다" 알림
2. 브라우저 창이 자동으로 열림
3. 다우오피스에 로그인 (3분 내)
4. 로그인 완료되면 자동으로 원래 작업 계속 진행

수동으로 갱신이 필요한 경우:
```bash
python login.py
```

### 브라우저가 안 열림

`config.yaml`에서 `headless: false`로 설정하면 브라우저 창이 보입니다.

### 로그 확인

로그 파일은 `logs/qw-agent-YYYY-MM-DD.log`에 저장됩니다.

```

├── src/
│   ├── main.py           # 진입점, 트레이 아이콘

│   ├── llm_parser.py     # 자연어 파싱 (Gemini)

│   └── config.py         # 설정 로드
├── config.yaml           # 설정 파일
├── cookies.json          # 다우오피스 로그인 쿠키
├── credentials.json      # Google OAuth 자격증명
├── login.py              # 로그인 스크립트
├── google_auth.py        # Google 인증 스크립트
└── logs/                 # 로그 파일
```

## 주의사항

- 다우오피스 세션 만료 시 자동으로 브라우저가 열려 재로그인 가능 (3분 내)
- 회의실 예약은 업무시간(09:00~18:00) 기준으로 해석됩니다
- "1시"는 오후 1시(13:00)로 해석됩니다
- 조회 기능은 11층 회의실만 표시됩니다
- Google Calendar 등록 옵션은 예약 모달에서 변경 가능
